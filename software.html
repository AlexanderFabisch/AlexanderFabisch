Software¶

Libraries¶

  • OpenANN: An open source library for artificial neural networks.
  • CMA-ESpp: A C++ implementation of the derivative-free optimization algorithm CMA-ES.
  • CPP-Test: Minimal C++ unit testing framework.
  • Smart-Script A short ruby class that saves lots of code for simple Linux scripts.

Scripts¶

  • GenImg: A small script that generates images (PNG oder PDF) for LaTeX documents. Supported formats are gnuplot scripts (ending .plot or .gnuplot), dot graphs (ending .dot), dia diagrams (ending .dia), PS, EPS und SVG. The Ruby gem Smart-Script is required. The script works only on Linux.
