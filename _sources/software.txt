Software
========

Libraries
---------

* `OpenANN <https://github.com/AlexanderFabisch/OpenANN>`_:
  An open source library for artificial neural networks.
* `CMA-ESpp <https://github.com/AlexanderFabisch/CMA-ESpp>`_:
  A C++ implementation of the derivative-free optimization algorithm CMA-ES.
* `CPP-Test <https://github.com/AlexanderFabisch/CPP-Test>`_:
  Minimal C++ unit testing framework.
* `Smart-Script <https://github.com/AlexanderFabisch/Smart-Script>`_
  A short ruby class that saves lots of code for simple Linux scripts.

Scripts
-------

* `GenImg <http://informatik.uni-bremen.de/%7Eafabisch/files/genimg.rb>`_:
  A small script that generates images (PNG oder PDF) for LaTeX documents.
  Supported formats are gnuplot scripts (ending .plot or .gnuplot), dot graphs
  (ending .dot), dia diagrams (ending .dia), PS, EPS und SVG. The Ruby gem
  Smart-Script is required. The script works only on Linux.
